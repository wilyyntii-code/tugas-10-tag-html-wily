# tugas-10-tag-html-wily
1.  <html> - Fungsi: Sebagai tag utama yang menandakan awal dan akhir dokumen HTML.
2.  <head> - Fungsi: Berisi informasi tentang halaman web seperti judul dan pengaturan, tapi isinya tidak tampil di layar.
3.  <title> - Fungsi: Untuk membuat judul yang muncul di bagian atas tab browser.
4.  <body> - Fungsi: Tempat meletakkan semua konten yang akan terlihat oleh pengunjung (teks, gambar, dll).
5.  <h1> - <h6> - Fungsi: Digunakan untuk membuat judul atau header.  <h1>  ukurannya paling besar, sedangkan  <h6>  paling kecil.
6.  <p> - Fungsi: Digunakan untuk membuat paragraf teks biasa.
7.  <a> - Fungsi: Membuat link atau hyperlink agar bisa diklik dan pindah ke halaman lain.
8.  <img> - Fungsi: Digunakan untuk memasukkan atau menampilkan gambar ke dalam web.
9.  <div> - Fungsi: Sebagai wadah atau kotak untuk mengelompokkan beberapa elemen agar mudah diatur.
10.  <br> - Fungsi: Membuat baris baru atau enter di dalam teks.
